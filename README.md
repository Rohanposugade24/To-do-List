# 📝 To-Do List App

A simple To-Do List web application built using HTML, CSS, and JavaScript with Local Storage support.

## 🚀 Features
- Add tasks
- Mark tasks as completed
- Delete tasks
- Save tasks using Local Storage

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📂 Project Setup
1. Download or clone the repository
2. Open the project folder in VS Code
3. Run `https://todolist-six-roan-65.vercel.app/` in browser

## Live Demo
https://todolist-six-roan-65.vercel.app/

## 💾 Local Storage
Tasks are stored in browser Local Storage, so data remains saved even after refreshing the page.
